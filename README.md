### Neovim Essentials

```
# Modes
i           # Insert mode (type text)
<Esc>       # Normal mode (give commands)
v           # Visual mode (select text)
:           # Command mode

# Movement
hjkl        # Left, down, up, right
w b e       # Word forward, back, end
0 $ ^       # Line start, end, first char

# Editing
dd          # Delete line
yy          # Copy line
p           # Paste
u           # Undo

# Files
:w          # Save
:q          # Quit
:wq         # Save and quit
```
