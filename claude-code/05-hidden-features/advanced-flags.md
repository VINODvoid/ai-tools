# Advanced Flags - Power User Options

**Complete reference for CLI flags**

---

## Essential Flags

### Model Selection
```bash
--model sonnet    # Default
--model opus      # Powerful
--model haiku     # Fast/cheap
```

### Permission Modes
```bash
--permission-mode default
--permission-mode acceptEdits
--permission-mode plan
```

### Tool Management
```bash
--tools "Bash,Read,Edit"
--allowedTools "Read,Grep"
--disallowedTools "Bash"
```

### Session Control
```bash
--continue
--resume [id]
--session-id <uuid>
```

### Output Formats
```bash
--output-format text
--output-format json
--output-format stream-json
```
